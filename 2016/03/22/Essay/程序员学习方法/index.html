程序员学习方法

发表于 2016-03-22 | 分类于 自由随笔
自己总结了一些程序员学习的方法，不定时更新……

自学

如今的时代，知识大爆炸，终身学习对人愈来愈重要，简要谈一谈我自己在实际工作中自学的方法。

学习方法

  1. 先把官方文档过一遍，不是为了明白，是为了熟悉全局
  2. 不要纠结一句话的不理解，也许后面三段都是解释这一句话
  3. 上手写 demo，一定要亲手敲打每一个 api
  4. 遇到 bug 再过一遍官方文档，寻找解决方法
  5. 解决不了就去查找第三方资源: github，大牛博客，书，Stack Overflow ，Google
  6. 然后理解总结别人为什么会这么写代码，配合再去看一遍官方文档，深入理解
  7. 一定要总结自己学过的知识(画脑图，写博客-开源精神)
  8. 先 what 然后 how 最后 why
