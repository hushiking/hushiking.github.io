Python list/tuple/dict/set

list

  • python内置的一种数据结构
  • 有序
  • 可更改（添加、删除）

tuple

  • python 内置的一种数据结构
  • 有序
  • 不可更改
  • 在赋值的时候决定所有元素

dict

  • python 内置的一种数据结构
  • 无序
  • 可更改

dict是典型的用空间换时间的例子。会占用大量内存，但是查找、插入速度很快，不会随着元素数量增加而增加。
list则是时间换空间的例子，不会占用大量内存，但是随着元素数量增多，查找时间会变很长。

set

  • python 内置数据结构
  • 无序
  • 可更改

set可以视为没有value的dict，只存key，一般用做去重或者集合求交、求并等。

  2. 叶子节点只存在最下面两层。

  4. 一个节点不可能只有右孩子。
  5. 一个节点的左孩子和右孩子都比这个节点大（或者小）(大顶堆或小顶堆)
