查看端口占用

window+R –> cmd –> 进入命令提示符

输入 netstat -ano 即可看到所有连接的 PID。之后在任务管理器中找到这个 PID 所对应的程序。如果任务管理器中没有 PID 这一项，可以在任务管理器中选“查看”-“选择列”。

  1. 查看所有端口占用情况

    C:\Users\Administrator>netstat -ano

  2. 查看指定端口号占用情况（也可输入PID）

    C:\Users\Administrator>netstat -ano|findstr “3000”

  3. 查看PID对应的进程（必须输入PID，否则查询不到对应端口号）

    C:\Users\Administrator>tasklist|findstr “6704”

  4. 结束该进程

    C:\Users\Administrator>taskkill /f /t /im node.exe

    /f表示强制终止进程

    /t表示终止指定的进程和任何由此启动的子进程

    /im name表示要终止的进程的映像名称，可以使用通配符*

    /PID process id指定要终止进程的PID

Mstsc (Microsoft terminal services client)

创建与终端服务器或其他远程计算机的连接

Mstsc还有一种说法，Microsoft Telnet Screen Control，即“微软远程桌面控制”。

操作方式: 打开 cmd，输入 mstsc 回车打开远程连接，当出来界面后，输入对方的IP和端口后，即可远程到对方的电脑，从而在本地操作对方的电脑。
